| MX Master 3  |     Yes     | `Wireless Mouse MX Master 3` |

|  MX Master   |     Yes     |                              |
| MX Vertical  |     Yes     |                              |

|     M720     |     Yes     |                              |
|     M590     |     Yes     |                              |
|     T400     |     Yes     |                              |
|   MX Keys    |     Yes     | `MX Keys Wireless Keyboard`  |
